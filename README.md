# HR Management System

Простое веб-приложение для ведения кадрового учёта сотрудников: регистрация, редактирование, просмотр анкеты, оформление отпусков и больничных, расчёт стажа

## Структура проекта

- `app.py` — backend-логика 
- `templates/` — HTML-шаблоны страниц (Bootstrap UI)
- `ddm.sql` — SQL-скрипт создания структуры базы данных (PostgreSQL)

## Технологии

- База данных: **PostgreSQL (Supabase)**
- Backend: **Python + Flask**
- Деплой: **PythonAnywhere**
